# Interface

* ``` note / Creates a new note ``` 
* ```note l / list [-c/--category "category"]```
* ```note rm / remove/ "note id [-c/--category "category"]"```
* ```note mv / move/ id category```
* ```note a / add "note" [-c/--category "category"]```
* ```note -h / --help```
